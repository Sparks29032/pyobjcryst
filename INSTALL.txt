Dependencies
-------------------------

The following programs must be installed on your system before you can install
pyobjcryst.

Python >= 2.5, < 3.0
 * http://www.python.org

python-setuptools >= 0.6c9
 * http://pypi.python.org/pypi/setuptools 

Boost C++ libraries >= 1.39
 * http://www.boost.org/

ObjCryst++ (see below)
 * http://vincefn.net/ObjCryst/
 * Newmat C++ matrix library > 10
   * http://www.robertnz.net/nm_intro.htm
 * cctbx installed as shared libraries
   * http://cctbx.sourceforge.net/


Installing pyobjcryst
-------------------------
To install pyobjcryst, you must have ObjCryst++ installed as a shared library
(see below). Once this is done, type the following from the command line :

> python setup.py install

For installation options, type :

> python setup.py --help install

Installing ObjCryst++
-------------------------
We provide scons build script and source bundle to make it easier to build
ObjCryst++ and its dependencies as a shared library. Here's how to install.

 1. Download ObjCryst-latest.tar.gz from http://dev.danse.us/packages/.
 2. Extract the archive to the 'libobjcryst' directory :
  
    > tar xzvf ObjCryst-latest.tar.gz -C libobjcryst
 
 4. Move into the libobjcryst directory and run the following command :

    > scons build=fast install

    Run `scons -h` for other installation options.

Once you have done this, you can install pyobjcryst as indicated above.
